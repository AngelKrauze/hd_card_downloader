## Description
Simple script that download high definition cards for your decks intead of the poor quality default ones.

<img src="https://images.ygoprodeck.com/images/assets/ygoprodeck_header_logo.png">

## Requirements
- [x] bash
- [x] curl

<sub>This project is written in POSIX so any shell will work.</sub>
## Usage
- Replace **`path`** inside the script  with your own path.
- Run it.
