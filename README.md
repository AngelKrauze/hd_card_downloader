### Description

Simple script that ACTUALLY download HD versions of your deck cards. Written in POSIX shell so its actually OS agnostic.

<img src="https://images.ygoprodeck.com/images/assets/ygoprodeck_header_logo.png">

### Requirements
- Any shell (BASH, DASH, etc.)
- cURL

### Usage
- Replace `$path` inside the script  with your own path.
- Run is as `sh hd_card_downloader.sh`
